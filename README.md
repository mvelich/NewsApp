# NewsApp

News application that shows news for the last 7 days from [newsapi.org](https://www.google.com)

## Build

Swift Package Manager is used to manage dependencies. To build the app just open it in Xcode and run on device.
###### Make sure that deployment target in Xcode lower than IOS-device version.

### Libs / Frameworks:
* [CoreData](https://developer.apple.com/documentation/coredata)
* [Kingfisher](https://github.com/onevcat/Kingfisher)
* [Alamofire](https://github.com/Alamofire/Alamofire)
